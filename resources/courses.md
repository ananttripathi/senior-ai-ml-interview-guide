## Free University Courses

### Machine Learning
**Stanford CS229 — Machine Learning**
- Instructor: Andrew Ng (legacy) or current faculty
- Best for: mathematical foundations — gradient descent, probabilistic models, SVM, EM algorithm
- Where: YouTube (full lectures), cs229.stanford.edu (notes)
- Relevant lectures: 1–10 (core ML), 17–20 (unsupervised learning)

**Caltech CS156 — Learning From Data**
- Instructor: Yaser Abu-Mostafa
- Best for: deep theoretical understanding of learning theory, VC dimension, bias-variance
- Where: YouTube
- Use this if you want to go deeper than CS229 on theory

### Deep Learning
**Stanford CS231n — Convolutional Neural Networks for Visual Recognition**
- Best for: backprop deep dive, CNNs, visualization, transfer learning
- Where: cs231n.stanford.edu, YouTube

**Stanford CS224N — Natural Language Processing with Deep Learning**
- Best for: NLP, sequence models, Transformers, modern LLM architectures
- Where: cs224n.stanford.edu, YouTube
- Assignments are excellent — do them

**Fast.AI — Practical Deep Learning**
- Instructor: Jeremy Howard
- Best for: top-down practical approach, great for building intuition fast
- Where: fast.ai (free)
- Do this before CS231n if you're newer to deep learning

### Statistics / Data Science
**MIT 18.650 — Statistics for Applications**
- Best for: hypothesis testing, confidence intervals, regression, Bayesian inference
- Where: MIT OpenCourseWare

## Paid Courses (Worth It)

### ML System Design
**Machine Learning System Design Interview** — Educative.io
- Cost: ~$30/month
- Best: specifically designed for ML system design interviews; 15+ worked case studies
- Direct prep material; do the case studies and practice explaining them out loud

**MLOps Specialization** — DeepLearning.AI (Coursera)
- Covers feature stores, training pipelines, model serving, monitoring
- Good for ML Engineer roles; covers the production side interviewers probe

### LLMs / AI Engineering
**LangChain & LLM Bootcamp** — Full Stack Deep Learning (free on YouTube)
- Best for: AI Engineer roles; covers RAG, agents, evaluation
- Very practical; goes from concept to deployed application

**Building with Claude / OpenAI APIs** — various (DeepLearning.AI short courses)
- Free, short (1–2 hours each)
- Topics: tool use, RAG, fine-tuning, function calling
- Good for refreshing LLM application patterns quickly

### Coding
**LeetCode Premium** — ~$35/month
- Access to company-specific question lists (Google, Meta, Amazon, etc.)
- Mock interview mode with timer
- Worth it in the last 4–6 weeks of prep

## Specialization Tracks

### For Data Scientists
1. Stanford CS229 (ML fundamentals)
2. Kohavi's A/B testing book (or coursera equivalent)
3. SQL practice on StrataScratch (company-specific questions)
4. Product analytics: Mode Analytics tutorial + Amplitude blog

### For ML Engineers
1. CS229 for fundamentals + CS231n for deep learning
2. Designing Machine Learning Systems (Chip Huyen book)
3. MLOps Specialization (Coursera)
4. LeetCode patterns (focus on medium/hard)
5. Distributed systems basics: Martin Kleppmann's "Designing Data-Intensive Applications"

### For AI Engineers
1. CS224N (NLP and Transformer foundations)
2. Full Stack Deep Learning (practical LLM applications)
3. DeepLearning.AI short courses (RAG, agents, fine-tuning)
4. Fast.AI (quick practical intuition builder)
5. LeetCode for coding bar
